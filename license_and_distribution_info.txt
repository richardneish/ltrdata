This directory and subdirectories contains source code for some small
utilities. It is mostly OS and network administration and debugging tools.

"posix" contains source code that compiles in Unix-like environments with
gcc or g++ compilers. There is also a Makefile available that has been tested
under Linux, FreeBSD and Cygwin.

"win32/c" and "win32/cpp" contains source code that compiles in Win32
environment. "win32/include" contains headers for the WinStructured library
that is required for most cpp sources to compile.

"win32/exe" contains a Makefile for compiling with VC or VC++ compilers with
Win32 Platform SDK.

Compiled binaries for Win32 are available on this server in the /utils
directory. The binaries may in some cases be of newer versions than the sources
available here. To get the compiled binaries of the same version as the source
available here, download from http://here.is/olof/files/ instead.
